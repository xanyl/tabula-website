Light Mode Adaptation

Theme tokens via CSS custom properties
Dark (default)
Backgrounds: navy/black gradient
Text: near-white on dark
Panels: frosted glass on dark
Glow effects: vibrant
Light
Backgrounds: subtle warm gray
Text: charcoal on light
Panels: crisp white cards, subtle shadows
Accent colors: same hues, adjusted saturation

Motion Strategy

Animation tokens — Tailwind + CSS
Element Animation Technique
Sections (on scroll) Fade up 24px, 400ms ease-out CSS animation with IntersectionObserver trigger
Buttons Scale to 0.98 on press, translateY(-1px) on hover Tailwind transition utilities
Cards Subtle border glow on hover Group hover + transition-shadow
Hero gradient Slow radial gradient drift (20s loop) CSS @keyframes on background-position
Process steps Left accent bar fills on scroll into view CSS transition + IntersectionObserver
Mobile menu Slide in from right, backdrop fade CSS transform + opacity transition
Dashboard sidebar Collapse/expand with content reflow CSS width transition

What Stays vs What Changes

Preserved from current site

  • Dark navy/purple atmosphere + gradient backgrounds
  • Geist + Geist Mono font pairing
  • Glassmorphism panel aesthetic
  • Indigo/teal/purple accent palette
  • Compact, confident copywriting tone
  • 4-step process and services structure

Improved in rebuild

  • Add light mode with matching design tokens
  • Scroll-triggered reveal animations (currently none)
  • Consistent spacing scale (8px grid system)
  • Better hover/active states across all interactive elements
  • Proper focus rings for keyboard navigation
  • Animated hero gradient
  • Responsive refinements (current breakpoints are decent but coarse)
