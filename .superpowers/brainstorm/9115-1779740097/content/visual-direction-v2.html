New Visual Direction — "Intelligence Emerging from Chaos"

No preservation of the old. Built from first principles, informed by the best in class.

1. Design Philosophy

The website itself is Tabula's first case study. It must feel like what Tabula sells: clarity carved from complexity, intelligence made visible, friction dissolved.

Old approach
Dark-only theme
Static page with text-heavy sections
3 unrelated accent colors
No scroll narrative
Glassmorphism as decoration
Dense copy blocks
New approach
Dark + light, section-aware
Visual storytelling via scroll
Single dominant accent + system palette
Each section tells one beat of the story
Transparency used ONLY where it serves meaning
Breathe. Less text. More impact.

3. Typography

Inter (UI) + Instrument Serif (contrast moments)
Where workflows become intelligent.
Instrument Serif for hero headlines and key pull-quotes — adds warmth and authority. Not decorative. Strategic.
Inter for everything else: body, navigation, cards, UI. Clean, neutral, highly readable at all sizes.
Font pairing rationale: Serif headline = "we think deeply." Sans body = "we execute precisely."

4. Spacing & Rhythm

4px base grid. Generous. Intentional.
Section padding: 120px top, 120px bottom
Card gaps: 16px (grid), 24px (stacked)
Content max-width: 1200px (marketing), 1400px (dashboard)
Hero min-height: 90vh (fill the screen)
Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64, 96, 120, 160

Every section gets ONE job. One message. Then white space.
No stacking 3 messages into one section to "save space."

5. Motion Language

Animation Purpose Tech
Hero: particles coalesce into the Tabula mark Visual metaphor: scattered → structured Canvas + requestAnimationFrame
Sections: staggered fade-up on scroll Content feels alive, responds to the user Framer Motion
Process: steps connect with an animated line Shows flow, progression, continuity Framer Motion SVG path
Numbers: count-up on scroll into view Makes metrics feel real, not static Framer Motion useSpring
Theme toggle: smooth color dissolve Polished, not jarring CSS transition on :root vars
Page transitions: subtle crossfade Navigation feels smooth, app-like Framer Motion AnimatePresence

6. Homepage Layout (Scroll Narrative)

Each scroll height = one complete thought
1. Hero (100vh) — Particles coalesce into the Tabula mark. One sentence. One CTA. That's it.
"Turn manual workflows into intelligent systems." → [See how] [Start a conversation]
2. The Problem (80vh) — Split screen. Left: chaotic, fragmented visual (scattered papers, emails, tools). Right: clean, connected system. This IS the Tabula promise, shown not told.
Before/After visual — no text needed beyond section label
3. How It Works (auto) — 4-step animated process. A line connects the steps as you scroll, following the Diagnose→Design→Build→Refine journey. Each step highlights as it enters viewport.
Interactive timeline, not a static grid
4. Services (auto) — Bento grid layout. 4 cards of varying sizes, not a uniform grid. AI Audit gets the hero slot (biggest card). The layout hierarchy communicates priority.
Asymmetric bento grid — like Linear's feature page
5. Social Proof (auto) — Case study metrics as large numbers (count-up animation). Client logos. One pull-quote testimonial. No carousel — just trust signals.
Metrics-first, logos-second. Numbers are the real proof.
6. CTA (80vh) — Dark → light section transition. Full-bleed indigo gradient. One question: "Ready to make your workflows intelligent?" — email input + submit. Form submits via server action.
Section transitions from dark to light background — signals a new beginning
7. Footer — Clean, minimal. Navigation, tagline, theme toggle. Nothing heavy.

7. Dashboard Layout

Strict, functional, clear — not decorative
